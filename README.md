## 🧠 Face Verification App using DeepFace & OpenCV

This is a real-time face verification system built in Python using OpenCV and DeepFace. It captures live webcam video and compares each frame to a reference image (`elon.jpg`) to detect a face match.

## 🚀 Features ##

- 🔁 Live webcam feed
- 🤖 Face verification using DeepFace
- 🧵 Threading for smooth performance
- ✅ On-screen "MATCH!" or "NO MATCH!" status
- ⌨️ Press `q` to quit the app
